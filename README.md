**Healthcare Funding Efficiency Analysis**

📌 Project Overview
The Healthcare Funding Efficiency Analysis project focuses on evaluating how healthcare funding aligns
with disease mortality across major healthcare research and disease areas. The project combines Tableau
dashboards, KPI-driven analytics, and data storytelling techniques to identify funding imbalances,
underfunded high-risk diseases, and healthcare resource allocation patterns.
Using interactive visualizations and custom analytical metrics, the project transforms raw healthcare
funding and mortality data into actionable business insights for strategic healthcare decision-making.

🎯 Project Objectives
- Analyze healthcare funding trends across disease areas
- Compare funding allocation with mortality impact
- Develop a custom KPI to measure funding efficiency
- Identify underfunded and high-risk healthcare areas
- Create interactive executive-level dashboards in Tableau
- Deliver data-driven healthcare insights through visualization
- 
📊 Key KPI Used
Funding Efficiency KPI

Formula:
Funding per Death = Total Funding ÷ Mortality (2019)
This KPI measures how healthcare funding is distributed relative to disease mortality impact.

Dashboard 2 — Funding Efficiency & Mortality
Analysis
Dashboard Features
- Funding Efficiency KPI
- Top 10 Funding Efficiency Areas
- Funding vs Mortality Scatter Plot
- Underfunded High-Risk Disease Identification
- Comparative Mortality Analysis
  
Key Insights
- Cardiovascular diseases show extremely high mortality levels.
- Cancer receives the highest funding allocation among major disease areas.
- Some diseases demonstrate disproportionately high mortality despite moderate funding.
- Funding efficiency varies significantly across healthcare research areas.

📌 Business Insights

- Comparative analytics can support more optimized healthcare resource planning.
